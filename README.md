# Football Hub v3

Vanilla JS + HTML + CSS. GitHub → Vercel ile yayınlanır.

## Modlar
- **CLUB / PLAYER / MANAGER** — Wordle stili kelime tahmin
- **LOGO** — Bulanık logo quiz
- **DAILY** ⭐ — Herkese aynı kelime, gün serisi (streak)
- **BLITZ** ⚡ — 60 saniyede mümkün olduğu kadar çok kelime
- **KARİYER** 📜 — Kulüplerden oyuncuyu bul
- **DÜELLO** ⚔️ — Link paylaş, ikiniz aynı 5 kelimeyi yarışın

## v3 Yenilikleri
- ⚔️ Düello modu (deterministik seed, URL paylaşımı)
- 📜 Kariyer modu
- 🏅 15 rozet (achievement) sistemi
- 🌗 Tema değiştirici (koyu/açık)
- 📊 Detaylı stats: kazanma %, düello, blitz rekoru
- 🎯 Genişletilmiş veri seti

## Çalıştırma
Sıfır build. `vercel --prod` veya local için:
```
npx serve .
```
